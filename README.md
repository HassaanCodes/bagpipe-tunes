# Bagpipe Tunes

This is a work-in-progress

It is my first attempt at making a full-stack application. I am using the PERN stack.

The finished application will display a list of bagpipe tunes and their music sheet links. Upon clicking 
any of the links, you will be taken to a page with the music sheets.

The goal here is for you to be able to easily find music sheets for bagpipe tunes you like.

# Enjoy learning and piping
